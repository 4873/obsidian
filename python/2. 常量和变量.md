# 注意事项

python中的变量和常量不需要事先声明

变量在使用前必须赋值，变量赋值之后才会被创建成功

# 理解赋值

例如：
a="jack"

python解释器做了两件事
1. 在内存中创建了 jack 这个字符串对象
2. 在内存中创建了一个叫 a 的变量，并把它指向 jack


如果 b=a ,则是将变量 b 指向了变量 a 所指向的数据 ，b 和 a 的地址是一样的
