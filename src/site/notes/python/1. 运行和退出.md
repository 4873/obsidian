1. 直接运行：win+r 打开运行窗口，输入 py 或者 python ，输入代码运行
使用exit()或者quit()可以退出

1. 运行脚本：打开脚本的文件夹目录，在地址栏输入 powershell ，打开后输入 python 脚本名.py
