1. 整数/整型 int
2. 浮点数 float
3. 复数 complex
# 要点
tips：
- 浮点数过长时必须使用科学计数法，把 10 用 e 代替
例如：
1.23×10^9   就是 1.23e9       
0.000012    就是 1.2e-5

- 复数可以用 a+bj 或者 complex(a,b)表示
a，b都是浮点型

# 计算

- 加减乘除( + - *  / )
- 取整除 ( // )
- 取余/取模 ( % )
- 幂  (\*\*)    pow(x,y)
- 计算绝对值  abs() 
- 数值取整  int(变量) 
- 四舍五入  roung(变量)
